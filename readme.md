### Section Manager
- Can Attach images as much as you want (Paste from Clipboard)
- Save to localStorage
- Export / Import (.zip)
- Auto adjust RTL/LTR languages
- Modal Images
- Resize Sections
- Font Family / Size
- Record Audio
- Record Video
- Drag / Drop Re-order
- File Attachment (.txt, .html, .md, etc)
- Project Tabs

### [Live Demo](https://ixjb94.github.io/section-manager/)
