### Section Manager
- Can Attach images as much as you want (Paste from Clipboard)
- Save to localStorage
- Export / Import
- Auto adjust RTL/LTR languages
- Modal Images
- Resize Sections
- Font Family / Size
- Record Audio
- Drag|Drop Re-order
